# service-framework
Service/Provider framework with plugin isolation

## Maven

```xml
<dependency>
    <groupId>com.daedafusion</groupId>
    <artifactId>service-framework</artifactId>
    <version>1.0</version>
</dependency>

<!-- For yml based configuration files -->
<dependency>
    <groupId>com.daedafusion</groupId>
    <artifactId>service-framework-yaml</artifactId>
    <version>1.0</version>
</dependency>
```
